# Vinkulierung (OR 685a–685b)

- **Formale Grundlage**: Statuten können Namenaktien der Zustimmungspflicht unterstellen (OR 685a). Rule `VINK-01` stellt sicher, dass bei Verweis auf Vinkulierung auch eine entsprechende Klausel markiert ist.
- **Kategorien**: Welche Aktien betroffen sind, muss transparent sein. Rule `VINK-02` fordert mindestens eine Kategorie-Angabe (z. B. "Namenaktien Klasse A").
- **Wichtige Gründe**: Ablehnung darf nur aus wichtigen Gründen oder wegen Erwerb für Rechnung Dritter erfolgen (OR 685b Abs. 2). Rule `VINK-03` verlangt, dass ein statutarischer Hinweis auf diese Gründe oder Policies vorliegt. Substantielle Prüfungen (z. B. Gleichbehandlung) erfolgen ausserhalb des Rule-Packs als Decision Template (Class J).
- **Decision Template**: Evidence Packs verweisen auf ein separates Entscheidprotokoll, das dokumentiert, weshalb ein konkreter Erwerb abgelehnt wurde.
